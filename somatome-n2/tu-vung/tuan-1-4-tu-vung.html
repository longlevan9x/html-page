１．お 母 ( かあ ) さんが 赤 ( あか ) ちゃんを 背中 ( せなか ) に（　おんぶ　）している。
Người mẹ cõng đứa con trên lưng.

２．サッカーボールを 足 ( あし ) で（　けった　）。
Đá quả bóng bằng chân.

３． 子 ( こ ) どもが 注意 ( ちゅうい ) をいやがって（　あばれている　）。
Đứa trẻ ghét bị lưu ý, đang nổi sùng lên.

４． 食 ( た ) べたあとの、 洗 ( あら ) い 物 ( もの ) を（　 流 ( なが ) し　）に 持 ( も ) っていきなさい。
Sai khi ăn xong, mang chén dĩa dơ vào bồn rửa đấy nhé.

５． 爪 ( つめ ) をかむのは（　よしなさい　）。
Hãy ngừng cắn móng tay đi.

６． 父 ( ちち ) に 大声 ( おおごえ ) で（　どなられた　）。
Bị bố quát xối xả.
